The following actor definitions have been defined for this implementation guide. 

- [GPCCMP Client](ActorDefinition-GPCCMPClient.html)
    - The GPCCMP Client is a system that can display the GPCCPM form, prepopulate it with patient health information retrieved from the GPCCMP Server, allow a user to fill out the form and save the form to the GPCCMP Server.

- [GPCCMP Server](ActorDefinition-GPCCMPServer.html)
    - The GPCCMP Server allows the GPCCMP Client to retrieve patient health information and writeback completed or partially completed forms.
