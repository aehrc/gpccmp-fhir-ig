The following actor definitions have been defined for this implementation guide. 

- [GP CCMP Client](ActorDefinition-GPCCMPClient.html)
    - The GP CCMP Client is a system that can display the GPCCPM form, prepopulate it with patient health information retrieved from the GP CCMP Server, allow a user to fill out the form and save the form to the GP CCMP Server.

- [GP CCMP Server](ActorDefinition-GPCCMPServer.html)
    - The GP CCMP Server allows the GP CCMP Client to retrieve patient health information and writeback completed or partially completed forms.
